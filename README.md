# DevOps-Project-End-to-End-Automated-CI-CD-Cloud-Deployment
AWS • Terraform • Jenkins • Docker • Kubernetes (EKS) • Ansible • Nexus

- Automated AWS infrastructure using Terraform (VPC, subnets, EKS cluster, IGW/NAT, EC2).
- Configured Nexus Repository Manager on EC2 using Ansible.
- containerized a 3-tier Microservices app (frontend, backend, database) using Docker.
- Pushed images to Nexus and deployed the full application stack to EKS.
- Created Jenkins pipelines for infrastructure provisioning, Nexus setup, Docker build/push, and application deployment.
